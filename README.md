# e_commerce_sales_analysis
This project presents a comprehensive SQL-based sales analysis performed using MySQL Workbench. The dataset used in this project represents e-commerce transactions and includes key attributes such as order date, customer name, city, product category, product name, quantity, price, total amount, payment mode, and order status. The goal of this project is to explore the dataset and extract meaningful business insights using structured query language (SQL).

The analysis begins with data exploration and understanding the structure of the dataset. Various SQL queries were written to examine the data, identify patterns, and validate data consistency. Core SQL concepts such as SELECT, WHERE, GROUP BY, ORDER BY, and filtering conditions were extensively used to manipulate and analyze the data efficiently.

A major focus of the project is on data aggregation and summarization. The GROUP BY clause was used to categorize data based on different attributes such as city, product category, and order status. ORDER BY and DESC were applied to sort the results, making it easier to identify top-performing cities, highest-selling products, and most profitable categories.

In addition, aggregate functions such as SUM, AVG, COUNT, MAX, and MIN were utilized to compute important metrics like total revenue, average order value, total number of transactions, and product-wise performance. These calculations helped in understanding overall business performance and identifying key trends in the dataset.

The project also includes analysis of customer behavior, such as preferred payment methods and purchasing patterns across different cities and categories. By analyzing these aspects, the project provides insights into which segments contribute most to revenue and how sales vary over time and location.

Furthermore, this project demonstrates the practical application of SQL for real-world data analysis. It highlights the ability to transform raw transactional data into structured insights that can support business decision-making. The queries written in this project are optimized for clarity and efficiency, showcasing strong foundational knowledge of SQL.

Overall, this project serves as a solid demonstration of data analysis skills using SQL. It reflects the ability to work with structured datasets, apply analytical thinking, and derive actionable insights that can be useful for businesses in understanding their sales performance and improving strategies.
